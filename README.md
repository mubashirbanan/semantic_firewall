## üîç Features

`semantic_firewall` offers several useful features:

- **Deterministic Semantic Fingerprints**: Generate clear, unique fingerprints for Go functions to verify code integrity.
- **Backdoor Detection**: Easily detect potential backdoors in your code.
- **Static Single Assignment Analysis**: Analyze your code without executing it, ensuring safety and reliability.
- **Scalar Evolution Verification**: Confirm that the transformations of your code maintain their intended behaviors.
